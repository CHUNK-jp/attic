Attic はローカルLLMを使ったパーソナル検索エンジン。 ファイル名を忘れても、内容の記憶だけで検索できる。 クラウドなし・サブスクなし・データはすべて手元に。

Features
🔍

自然言語で検索

「田中さんに送った提案書」「Pythonのasync処理のメモ」など、記憶のままで検索できる。

🏠

完全ローカル動作

Ollama経由でLLMをローカル実行。ファイルは一切外部に出ない。ネット接続も不要。

⚡

インクリメンタル更新

初回以降は変更ファイルだけ再インデックス。2回目から大幅に高速化される。

📄

多形式に対応

PDF・Word・Markdown・コード・JSON・CSVなど主要な形式をすべてカバー。

🆓

永久無料・OSS

MITライセンスのオープンソース。サブスクも課金もない。フォーク歓迎。

🎯

重複排除された結果

同一ファイルの複数ヒットを自動整理。ファイルごとにベストスコアだけを表示。

How It Works
ファイル → テキスト抽出 → チャンク分割 → Ollama埋め込み → ChromaDB
埋め込みモデルは nomic-embed-text（Ollama経由でローカル動作）。 ベクトルDBは ChromaDB（~/.attic/db/ に保存）。 検索クエリも同じモデルで埋め込み、コサイン類似度でマッチングする。
Quick Start
01

Ollama をインストール・起動

ollama.com からインストール後：

ollama serve
ollama pull nomic-embed-text
02

依存パッケージをインストール

pip install -r requirements.txt
03

フォルダをインデックス化

python attic.py index ~/Documents
04

自然言語で検索

python attic.py search "去年の確定申告の資料"
python attic.py search "Q1 budget report"
python attic.py search "Pythonのasync処理のメモ"
